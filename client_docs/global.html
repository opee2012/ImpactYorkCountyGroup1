Dashboard() → {JSX.Element}

The Dashboard component serves as the main interface for users and staff. It provides navigation through a sidebar, user actions in a top panel, and a content section for displaying data. It uses various hooks for authentication, data fetching, and state management. For users, the dashboard allows viewing data, searching, and filtering through the data. For staff, the dashboard provides the ability to log in and keep the data updated or make changes as needed. The dashboard also handles automatic data refresh in case of errors and provides a logout functionality.
